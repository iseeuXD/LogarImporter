
# Logar Importer

Logar ULP to MySQL aracı. İyi kullanımlar dilerim.




## Kurulum

Nasıl Kurulur?

```bash
  git clone https://github.com/iseeuXD/LogarImporter.git

  pip install -r requirements.txt

  python logar.py
```
